## 五、页面状态变体清单

每个核心页面需要实现以下状态变体：

```markdown


## 登录页（login）
- [ ] 正常态：角色切换器、账号密码输入、登录按钮
- [ ] 角色选中态：4 种角色的选中样式差异
- [ ] 账号停用 Dialog："账号已停用，请联系管理员"

## 首页（home）
- [ ] 正常态：渐变横幅、统计卡、快捷操作、列表
- [ ] 加载态：Skeleton 占位
- [ ] 空态：暂无数据（首次使用）

## 会议列表（meetings）
- [ ] 正常态：筛选 Tab + MeetingCard 列表 + FAB
- [ ] 加载态：Skeleton 列表
- [ ] 空态：暂无会议 + 引导创建
- [ ] 无网络态：顶部横幅 + 禁用 FAB

## 开始会议录音（recording）
- [ ] 正常态：表单 + 录音控制区
- [ ] 录音中态：计时器 + 声波动效 + 暂停按钮
- [ ] 无网络态：录音按钮禁用 + Toast
- [ ] 网络中断态：提示"网络异常，录音结束后需联网上传"

## 上传录音（upload-recording）
- [ ] 未选择态：文件选择卡
- [ ] 已选择态：文件信息卡
- [ ] 上传中态：进度条 + 速度 + 剩余时间
- [ ] 上传失败态：失败提示 + 重试按钮
- [ ] 格式错误 Toast
- [ ] 大小超限 Toast
- [ ] 无网络 Toast

## AI 处理中（processing）
- [ ] 正常处理态：Stepper 逐步完成
- [ ] 处理中态：某一步显示"处理中...预计 X 分 X 秒"
- [ ] 失败态：API 超时/限流/网络异常 + 重试按钮
- [ ] 无网络态："网络异常，转写将在网络恢复后继续"

## 审核会议纪要（ai-review-transcript）
- [ ] 正常态：会议信息 + 播放器 + 转写文本 + 操作按钮
- [ ] 低置信度态：红色提示条 + 置信度标签
- [ ] 驳回确认 Dialog
- [ ] 无权限态：NoPermissionState

## 审核任务列表（ai-review-tasks）
- [ ] 正常态：AI 摘要卡 + 任务列表 + 批量操作
- [ ] 任务编辑 BottomSheet
- [ ] 发布确认 Dialog
- [ ] 无权限态：NoPermissionState
- [ ] 缺少执行人 Toast
- [ ] 缺少截止时间 Toast

## 会议溯源（meeting-trace）
- [ ] 正常态：会议概要 + 文件留存 + 溯源时间线 + 关联任务
- [ ] 加载态：Skeleton

## 任务列表（tasks）
- [ ] 正常态：筛选 Tab + TaskCard 列表
- [ ] 加载态：Skeleton 列表
- [ ] 空态：暂无任务
- [ ] 临期标识：橙色左边条
- [ ] 逾期标识：红色左边条

## 任务详情（task-detail）
- [ ] 正常态：任务信息 + 附件 + 进度时间线 + 操作按钮
- [ ] 执行人员态：显示"附件、进度、完成"操作按钮
- [ ] 审核人员态：仅查看，无操作按钮
- [ ] 提交完成确认 Dialog

## 提交进度（progress-submit-sheet）
- [ ] 正常态：进度滑杆 + 说明输入
- [ ] 未填写 Toast
- [ ] 超出范围 Toast

## 附件上传（attachment-upload-sheet）
- [ ] 正常态：上传入口 + 附件列表
- [ ] 上传中态：进度条
- [ ] 上传成功态：勾选图标
- [ ] 上传失败态：重试按钮
- [ ] 单文件超限 Toast
- [ ] 总量超限 Toast

## 管理后台（admin）
- [ ] 组织架构 Tab：部门树 + 部门卡片
- [ ] 账号管理 Tab：账号列表 + 筛选
- [ ] 删除部门确认 Dialog（有关联人员时阻止）
- [ ] 无权限态：NoPermissionState

## 消息中心（messages）
- [ ] 正常态：分类 Tab + 消息卡片列表
- [ ] 全部已读态：清空红点
- [ ] 空态：暂无消息

## 我的（profile）
- [ ] 正常态：用户卡 + 权限说明 + 功能入口 + 角色切换 + 退出登录
- [ ] 角色切换演示态：4 个角色选项

## 个人备忘（memo）
- [ ] 正常态：今日提醒卡 + 备忘列表 + FAB
- [ ] 空态：暂无备忘 + 引导创建
- [ ] 到点提醒弹窗：类似微信消息提醒样式

## 备忘编辑（memo-edit-dialog）
- [ ] 正常态：标题 + 内容 + 提醒时间
- [ ] 提醒时间早于当前 Toast
```
